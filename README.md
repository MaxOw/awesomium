# Awesomium Bindings #

High-level bindings to [Awesomium](http://www.awesomium.com)(1.6.5)
for Haskell.

You will need [awesomium-raw](https://github.com/MaxOw/awesomium-raw)
library to use this.

For usage example go [here](https://github.com/MaxOw/awesomium-example)
